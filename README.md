# Kevin's Game Hub

一个轻量网页游戏集合平台，包含反应、记忆和策略类小游戏。

## 项目结构

```text
kevin-s_game/
├── index.html
├── styles.css
├── README.md
└── games/
    ├── neon-runner/
    ├── memory-match/
    └── sokoban/
```

## 游戏列表

| 游戏 | 类型 | 说明 |
|------|------|------|
| 霓虹街机跑酷 | 跑酷 | 躲避障碍，收集金币，维持连击 |
| 记忆卡片配对 | 益智 | 翻转几何卡片配对图案 |
| 推箱子 | 策略 | 将所有方块推到目标点 |

## 快速开始

直接用浏览器打开 `index.html` 即可，或使用本地服务器：

```bash
python3 -m http.server 8000
```

访问 `http://127.0.0.1:8000`。

## 操作说明

### 霓虹街机跑酷

- 键盘：`A` / `D` 或方向键左右移动
- 触屏：拖动角色移动
- 护盾能挡一次撞击，连续吃金币提升连击分

### 记忆卡片配对

- 点击或键盘聚焦后确认来翻转卡片
- 使用“预览”可以短暂查看未匹配图形
- 记录步数和时间

### 推箱子

- 键盘：方向键或 `W` / `A` / `S` / `D` 移动
- 触屏：使用页面方向键移动
- 支持撤回一步和重开当前关卡

## 技术栈

- HTML5 + CSS3 + JavaScript
- Canvas API
- Web Audio API
- 响应式布局
