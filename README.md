# 🎮 Kevin's Game Hub

一个网页游戏集合平台，包含多款有趣的小游戏！

## 📁 项目结构

```
kevin-s_game/
├── index.html          # 游戏平台首页
├── styles.css          # 全局样式
├── README.md
└── games/
    ├── neon-runner/    # 霓虹街机跑酷
    └── memory-match/   # 记忆卡片配对
```

## 🎯 游戏列表

| 游戏 | 类型 | 说明 |
|------|------|------|
| 霓虹街机跑酷 | 跑酷 | 躲避障碍，收集金币 |
| 记忆卡片配对 | 益智 | 翻转卡片配对图案 |

## 🚀 快速开始

### 本地运行

直接用浏览器打开 `index.html` 即可，或使用本地服务器：

```bash
python3 -m http.server 8000
```

访问 http://127.0.0.1:8000

### 部署到 GitHub Pages

1. 确保代码已推送到 GitHub 仓库
2. 进入仓库 Settings → Pages
3. 在 Source 中选择：
   - Branch: `main`
   - Folder: `/ (root)`
4. 点击 Save，等待几分钟即可生效

## 🎮 操作说明

### 霓虹街机跑酷
- **键盘**: A/D 或方向键左右移动
- **触屏**: 拖动角色移动
- 护盾能挡一次撞击
- 连续吃金币提升连击分

### 记忆卡片配对
- 点击翻转卡片
- 找到所有配对的图案
- 记录步数和时间

## 🛠️ 技术栈

- HTML5 + CSS3 + JavaScript
- Canvas API（跑酷游戏）
- 响应式设计
- Web Audio API（音效）
